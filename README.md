# 数电发票勾选整理台

本地网页工具。用途是把你手里的 `PDF` 数电发票，与“全量发票查询导出结果.xlsx”进行匹配，再按后台固定保存的“抵扣勾选增值税发票信息.xlsx”模板生成最终上传文件。

## 当前逻辑

1. 上传当月需要勾选的 `PDF` 发票，包括增值税专用发票和员工提交的铁路电子客票
2. 上传“全量发票查询导出结果.xlsx”
3. 后台自动读取 `samples\抵扣勾选增值税发票信息.xlsx` 模板
4. 系统优先按 `数电发票号码` 匹配
5. 若某张专用发票 PDF 在全量导出结果中找不到对应记录，直接报错
6. 若上传的是增值税普通发票，只统计数量与留档，不写入勾选清单
7. 成功匹配后，按模板列顺序生成新的勾选上传文件
8. “生成导入清单模板”会自动读取 `samples\导入清单模板.xlsx`，把上月未上传专票改写成导入模板格式
9. 铁路电子客票只有员工上传 PDF 且在全量导出结果中匹配成功时才写入勾选清单；全量表中未上传的铁路电子客票不会进入导入清单模板
10. 部分红冲专票的有效抵扣税额按“原发票有效抵扣税额 - 累计红冲有效抵扣税额”计算；全额红冲发票不写入勾选清单

## 启动方式

```bash
npm install
npm start
```

然后在浏览器打开 `http://localhost:3000`

## 当前已处理的关键字段

- 开票日期：以“全量发票查询导出结果.xlsx”为准
- 模板必填项：
  - 是否勾选*
  - 数电发票号码
  - 开票日期*
  - 金额*
  - 票面税额*
  - 有效抵扣税额*
  - 购买方识别号*
  - 票种*
